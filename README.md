 # Task Manager Application

This is a simple task management application written in Python. It allows users to:
- Add tasks.
- View all tasks.
- Mark tasks as completed.

## Features
- Add a task with a title and description.
- View all tasks with their current status (Pending or Completed).
- Mark tasks as completed.

## Requirements
This application does not have any external dependencies, so you just need Python 3.x installed.

## How to Run
1. Clone the repository or download the `task_manager.py` file.
2. Open a terminal/command prompt.
3. Navigate to the folder where `task_manager.py` is saved.
4. Run the script using the following command:

    ```bash
    python task_manager.py
    ```

## Usage
Once the application starts, you will be presented with a menu:

1. **Add Task**: You will be prompted to enter a task title and description.
2. **View Tasks**: Displays a list of all tasks with their current status.
3. **Mark Task Completed**: You will be prompted to enter the title of the task you want to mark as completed.
4. **Exit**: Exits the program.
